## **Let Everyone Fully Own Their Social Memory**

Your chat history shouldn't be held hostage by platforms, lost to broken devices, or washed away by time.

ChatLab's mission is to build a privacy-first, platform-agnostic chat memory analysis infrastructure — giving users absolute control over every word they've ever said.

## What It Can Do Now

- Import chat history from WhatsApp, Telegram, Discord, iMessage, and more
- Talk to your chats — query any historical content in natural language
- Visual insights: relationship analysis, member activity, time distribution, topic tracking, and more
- Fully offline — your data never leaves your device

## What's Next

### More Platforms: A More Complete Social Memory

Your social memory is scattered across countless apps — chats, DMs, posts, comments. We're building adapters to cover more platforms, so that for the first time, you can see a complete and coherent picture of yourself across every single one of them.

### AI Agents: Conversations That Truly Understand You

Today's AI can answer "What did they say?" Next, we're building:

- Relationship network: Automatically build your relationship network. Who's closest to you? How many circles do you run in? Who's the real leader of the group?
- Memory nodes: Automatically find important decisions, emotional turning points, and milestone events in conversations
- Emotional trajectories: How the warmth of a relationship changes over time, visualized
- Cross-platform identity merging: The Alex on WeChat, @alex on QQ, and @alex on Douyin — they're actually the same person

### Open Standards: Making Chat Data Truly Interoperable

Today's chat data is a collection of islands — every tool exports differently, nothing is compatible.

ChatLab aims to drive the creation of an open standard for chat data, covering multiple layers:

- Universal record format — no matter which platform the data comes from, it can be converted to a unified structure without losing any information
- Universal import spec — any tool, any platform, following this spec, can import in one click
- Universal analysis format — relationship graphs, sentiment analysis, topic summaries... all analysis results have standardized data structures that any external tool can consume directly

We want this standard to become the common language of chat data — open, free, and interconnected — any developer can reuse it.

### CLI Engine: The Social Memory Primitive for the AI Era

Your chat history holds the most authentic version of you — your relationship network, decision patterns, emotional trajectory.

And the AI assistants of the future need to understand you to truly help you.

ChatLab CLI aims to become the standard interface for this social memory layer.

When Claude, OpenClaw, Hermes and other agents need to answer "What's my relationship with this person?", "Have we discussed this topic before?", or "Help me organize all important conversations with Alex" — they can call ChatLab directly to get structured, private, fully local memory data.

## Where Does This End?

We want ChatLab to become the infrastructure for chat data analysis.

Chat records from any platform come in, any analysis need is met, any user can use it.

Someday, your social memory will become the way you understand yourself best.
